혹시나 코딩에 익숙하지 않거나 처음이여서 이해하기 힘드신 분은 추가로 글을 남겨 주세요.

 

전체적인 코드는 간결하게 구성되어져 있습니다.

전체적인 동작 순서들을 옅은 사각형과 글자로 적어 두었습니다.

아래의 세부적인 기능에 대한 설명은 동작 순서에 맞게 진행 하겠습니다.

 

1. 버튼 및 음성 입력 버튼 동작 코드 (파란색 영역)

-- 첫번째 블럭은 음성 인식버튼을 누르게 되면 음성 인식 서비스를 실행하고 결과 값을 바로 아래에 있는 블럭을 통해 내용을 입력합니다. 이후 translate 함수를 실행합니다.

-- 마지막 3번째 블럭은 translate 함수만 실행 합니다. 음성 인식을 하지 않고 글을 복사하여 붙여넣거나 직접 내용을 입력하여 번역 하고 싶을때를 위해 남겨놓은 버튼 입니다.

 

※ 함수는 간단히 말해 여러개의 코드를 하나로 묶은 코드의 덩어리 입니다.

이때 반복되는 여러줄의 코드 또는 특정 기능을 수행하는 코드들을 함수로 깔끔하게 만들어두면 코딩할때 편리합니다.

 

 

2. 번역 시작 코드(함수)

2개의 버튼을 누르면 결과적으로는 translate 함수를 실행하게 됩니다.

그러나 2개의 버튼중에는 입력 값이 한글인지 영어인지 선택하는 버튼은 없습니다.

그렇기에 번역을 진행하기 전에 한/영 분별(Language_Detect 함수)을 먼저한 이후에 이에따른 번역 요청을 하게 됩니다.

 

※ Language_Detect 함수를 보면 입력 받는 함수가 있는데 이건 향후 제가 다른 곳에도 쓸때 편리하게 쓰기위해 저렇게 만들어 둔 것 입니다. 단지 입력된 문자값을 함수 동작하는 과정에 값을 읽는것이 아니라 함수 실행할때 같이 값을 넣어서 실행한다는 것 뿐 입니다.

2-1 한/영 분별

한/영 분별 방식은 한글과는 다르게 영어에만 있는 특성인 대/소문자 특성을 이용하여 한글인지 영어인지 판단합니다.

Language_Detect 함수 블럭에 보면 upcase와 downcase가 있습니다. 이는 앱 인벤터에 있는 기능으로 모든 문자를 대문자(upcase)로 바꾸거나 소문자(downcase)로 바꾸어 줍니다.

만약 한글만 있다면 대/소문자로 변환하여도 기존 원본과 차이점이 없기에 한글로 인식 할 수 있고 만약 하나라도 영어가 포함되어 있다면 대/소문자로 변환 할 경우 원본과 차이점을 보이게 됩니다.

이러한 원리를 통해 한/영을 판별하게 되며 이런 판별 값을 전역 변수인 language 변수에 저장하게 됩니다.

 

 

3. 번역 끝나면 음성 출력

한/영 판단 값을 통해 번역 요청에 필요한 코드(ko-en, en-ko)를 생성 합니다.

한글로 판단 되었다면 ko-en 영어로 판단 되었다면 en-ko로 지역변수인 lang2lang에 저장하고 이 변수를 번역 요청시 사용하게 됩니다.

 

번역 요청을 보내고 요청에 대한 응답(결과)이 도착하면 파란색 영역에 있는 함수(YandexTranslate1.GotTranslation)를 실행하게 됩니다.

결과 값을 TextBox(Output_Text)에 출력하고 TTS를 실행 하는데 이때 TTS를 실행 할 때 한/영 탐지 값과는 반대의 언어로 즉, 번역된 언어에 맞게 반대로 바꾸어 음성 출력을 합니다.

-- 영어에서 한국어로 번역 된것을 읽는데 한/영 탐지 결과에서 영어라고 된것을 그대로 가져와서 사용한다면 글을 읽는데 문제가 생기겠지요?

 

부록: 동기화(SYNC)문제점 발생 예시

Do라고 되어 있는 블럭은 제가 만든 앱에서 사용한 방식입니다.

Don't라고 되어 있는것은 단순히 보면 번역 요청을 하고 요청된 값이 TextBox에 입력되고 TTS가 잘 읽어 줄 것 처럼 보입니다.

하지만 요청을 보내는건 요청 보내고 나면 요청을 보내는 것으로서 동작이 끝나기에 응답(결과)이 오는것을 기다리지 않고 바로 그다음 함수를 실행하게 됩니다.

 

이렇게 되면 만약 번역 결과가 오기도 전에 글을 읽게 한다면 읽을 내용이 없어서 TTS 기능은 정상적으로 동작하지 않게 됩니다.

 

그래서 이러한 문제점을 방지 또는 해결하고자 한다면 응답을 받았을때 TTS가 실행되도록 Do라고 되어있는 블럭처럼 코드를 구성해야 합니다.
